Sold Unit Display for Woocommerce is the plugin to show total product sold number and unit (for purchased products) on your Woocommerce shop page and single product page.

== Description ==

This plugin is simple and strength forward to use.
It can display total product sold number like 'Sold x units' on Woocommerce shop.
This plugin also can display total product sold plus unit on Woocommerce single product page also.

Sold Unit Display for Woocommerce has the following options to configure.

*   Enable/Disable on Single Product Page
*   Enable/Disable on Product List/Archive Page
*   Hide if product has zeros sales
*   Background Color and Text Color control
*   Margin and Padding Control
*   Work with latest woocommerce version 5.4.x.

Any suggestion or feedback are always welcome.

== Frequently Asked Questions ==

= Which version of woocommerce that i can use with this plugin ? =

We tested with latest version of Woocommerce (5.5.x). However, you can also use with older version of woocommerce.

= How can i setup the plugin or manage plugin options ? =

Go to menu Woocommerce Settings under Products tab in section 'Unit Sold Display'.

= Why number does not updates ? =

Normally it would update along with the cache system expired. You can try clear your system cache to see the update. We does not use realtime ajax to read database since it is performance hoc.
